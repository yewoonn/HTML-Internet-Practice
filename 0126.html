Today I learned about several things in class.

First, I learned how to put link in my web page. It needs to write [a]. We can add some information such as title and reference by specific keyword.

Second, I found that there is parent and child tag. For example, there is [li] and [ol, ul]. [ol, ul] is parent tag for child tag [li]. We can classify the group of lists when we use parent tag and child tag. SO, WHAT IS DIFFERENT BETWEEN [ol, ul] ? [ol] is ordered list. we can put number in front of subject. On the other hand, [ul] is just unordered list. So, we can put a dot in front of subject.

Lastly, I learned SUPER STAR TAG. One of them is [head] tag that groups information of documentary. It includes [title] tag which shows title and [meta] tag which informs how to read this documentary. If we use Korean, we should write "utf-8" type. Also, there is [body] tag that groups all of the content in documentary. Above all there is the highest tag that groups head and body. It is [html] tag !!!
