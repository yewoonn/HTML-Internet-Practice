2. JS Keyword

  • HTML과의 차이

    • : HTML은 정적 언어로, 1+1을 입력하였을 때 있는 그대로 출력한다. 이와 달리 Java Script는 동적 언어로 1+1을 입력할 때 해당 식의 계산 결과인 2를 출력한다. 또한, 프로그램은 순서가 있는 명령어들의 집합으로 정의한다. HTML은 프로그램을 만드는 언어가 아닌 단지 화면에 표시할 내용을 가공하는 역할을 하므로 프로그래밍 언어라 지칭할 수 없다. 하지만, Java Script는 사용자와의 상호작용을 할 수 있도록 일종의 순서가 존재하는 명령어들이 존재하며 이는 프로그래밍 언어에 속한다 할 수 있다. 그러므로, Java script에는 객체, 함수, 반복문, 조건문 등 여러 개념들이 함께 사용된다.

  • script 태그

    • : Java Script 형식의 언어임을 표시하는 HTML 태그

  • 이벤트(event)

    • : 웹브라우저 상에서 마우스, 키 등의 움직임이 있을 때 사건을 발생시키는 것

    1. onclick - 해당 버튼을 클릭했을 때
    2. onchange - 해당 내용의 변화가 있을 때
    3. onkeydown - 어떠한 키가 눌렸을 때

  • QuerySelector

    • : 사용자와의 상호작용에 따라 특정 태그를 찾아 속성을 바꾸어 주어야 하는 상황에 처한다. 이를 위해 QuerySelector을 이용한다. document.querySelector('태그 이름')을 사용하면 찾고자 하는 태그를 지칭할 수 있다. 해당 태그의 하위 속성에 대해서는 '.'을 통해 연결하면 된다. 하지만, 이는 존재하는 첫 번째 태그만을 가져온다. 만약 모든 태그에 대해서 변화를 주고 싶다면, querySelectorAll을 사용한다. 이는 해당하는 모든 태그를 리스트 형태로 가져와 반복문을 사용하여 해결할 수 있다.
