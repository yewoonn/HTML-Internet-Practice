Today I learned about CSS in class.

First, I heard that how to develop my coding skill. It is removing duplication. When we remove duplication, it is more easy to read and debug.

Second, I learned how to change the color with CSS. One way is using [style] tag. [style] tag informs to web browser that this is CSS language. This is necessary because web browser try to read in HTML language basically. We should write a selector which is the tag we want to change color and color information inside the block. We can change color what we want!

The other way is using [style] property. We should put style property inside the [a] tag. When we use [style] property, we don't have to add selector. Because we already use this in specific tag.

Let's check some terms that we used. Selector is a tag name that we want to add design. We put Declaration inside the Selector block. In the Declaration, there is property like color and value like red.

Lastly, There are three kinds of selectors. One is tag selector, another is class selector, the other is id selector.

  1. Tag selector
    2. This is the widest selector. It refers to specific tags. You can change properties if the tag was used. Also, this is the weakest selector between other selectors. For example, There is one more class selector that using tag selector, it influenced by class selector.
  2. Class selector
    3. This is the selector that we can change properties in same class. We can add several tags to same class. We use [.] to call class selector.
  3. ID selector
    4. This is the smallest selector that we can change properties that have only the specific id. We can add only one tag to same ID. Also, this is the strongest selector between other selectors. We use [#] to call class selector.
