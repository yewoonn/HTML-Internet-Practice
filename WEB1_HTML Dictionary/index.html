  2. HTML Keyword

    • !doctype html

      • : 웹페이지 브라우저에게 html 버전 제시

    • p

      • : 단락, 태그 내부에서 Enter, Space를 하더라도 한 줄로 표현
      (p 태그 내에서 띄어쓰기를 하기 위해 pre 태그 이용)

    • pre

      • : 자유롭게 Enter, Tab, Space를 하기 위한 태그

    • h1~h6

      • : 크기에 따른 제목 글씨체, 숫자가 커질수록 글씨가 더 커진다.

    • li

      • : 리스트 형태, ol과 ul의 내부에서 사용되는 자식 태그

    • ul

      • : 정렬되지 않은 리스트 형태로 목록 제시, li의 외부에서 사용되는 부모 태그

    • ol

      • : 정렬된 리스트 형태로 자동으로 숫자 목록 제시, li의 외부에서 사용되는 부모 태그
