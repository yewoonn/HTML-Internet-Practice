2. CSS Keyword

  • style 태그

    • : CSS 형식의 언어임을 표시하는 HTML 태그

  • 선택자(selector)

    • : CSS로 디자인할 대상에 해당하는 부분

    1. Type 선택자 - html의 태그 이름에 해당하며 우선순위가 낮다.
    2. Class 선택자 - html의 클래스 이름에 해당하며 여러 개의 요소에 같은 이름의 클래스를 지정할 수 있다. '.'을 통해 표시한다.
    3. ID 선택자 - html의 고유 이름에 해당하며 다른 요소에 동일한 이름을 부여할 수 없다. '#'을 통해 표시한다.

  • display 형태

    • : 화면에 요소를 어떻게 드러나게 할지 결정하는 속성
    (각 태그에 알맞은 display 형태가 기본적으로 지정된다.)

    1. None - 화면에서 보이지 않는다.
    2. Block - 가로 한 줄을 모두 차지한다.
    3. Inline - 요소의 크기만큼만을 차지한다.

  • div

    • : 영역을 구분해주는 역할을 하며 block 형태로 지정된다.

  • span

    • : 영역을 구분해주는 역할을 하며 inline 형태로 지정된다.
