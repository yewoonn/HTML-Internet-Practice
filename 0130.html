Box Model in CSS

Box model is composed of several properties. You can control the space between word and border, called padding. Also, you can change the space between other blocks. It is called margin. Lastly, you can set width and height with block.

I learned 2 types of display. Inline is the display type that have only their size. On the other hand, block is the display type that have more than their size like one line. Actually all of the tags have default selection of display. But we can change display type with CSS.

There is some useless information but it is important for design. How to handle these information? We can use [div] and [span] tag for classify. [div] is block element that use all line. [span] has same function with [div] but it is just inline element.
